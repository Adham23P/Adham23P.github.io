BikeBuddy: Senior Year Capstone

Use of: Python, React-Native, AWS (S3, API Gateway, Lambda)

Enlarge/Enhance Enlarge/Enhance

(TO BE UPDATED) Mobile application that focuses on routes specifically for bikes in Washington, D.C. Python OSMNX library was used to get a base graph representation of the city. API calls to OpenData D.C. Datasets are the utilized in the creation of the graph and weight modificaiton for bike routes returned from out AWS lambda function. Bike lane information is retrieved from OpenData D.C. and bike lanes are overlayed onto our graph which is further reduced to nodes at each intersection. Straggler Mitigation

Use of: Python, Docker, AWS (EC2)

Enlarge/Enhance

Distributed ML training using PyTorch and RingAllreduce architecture. Stragglers were simulated and cut out based on varying conditions of completion and its lag compared to other workers. After aggregation occurs and each local model is updated, the straggler is added back into the ring architecture and training from each worker continues as normal. Performance was increased with minimal model degradation from dropped gradients from stragglers with credit card data set.

Microkernal Containers

Use of: C

Enlarge/Enhance

Implementation of containers in xv6 (microkernel operating system). Involved using shared memory with process mutex locks to communicate between a server and a client for starting a container. Involved modifications on both kernel and user-level with new system calls created to manage processes and sensitive actions.
